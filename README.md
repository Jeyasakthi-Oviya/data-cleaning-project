# data-cleaning-project
Data cleaning and preprocessing of a dataset using Python and Pandas
This project demonstrates data cleaning and preprocessing of a raw dataset using Python and Pandas. It also includes basic visualizations using Seaborn to explore the data.

## Steps performed:
- Handling missing values
- Removing duplicates
- Correcting data types
- Basic data visualization with Seaborn
- Exporting cleaned dataset

## Tools used:
- Python
- Pandas
- Jupyter Notebook
- Seaborn
